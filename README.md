# 🔮 Tarot Oráculo - Tiragem de Cartas Web

Uma aplicação web criativa, interativa e misteriosa para realizar sorteios de cartas de Tarot virtuais e revelar pequenas mensagens.

## 🌟 Objetivo do Projeto

A ideia aqui foi treinar a interatividade da página. O usuário acessa, clica para consultar o oráculo e, por baixo dos panos, o sistema embaralha uma série de cartas estruturadas no código e devolve uma resposta completamente aleatória (única) a cada clique.

## ✨ Destaques da Navegação

- **Sorteio Matemático:** Uso de funções lógicas do JavaScript para garantir um resultado imprevisível a cada botão pressionado.
- **Interatividade Dinâmica:** A página atualiza a carta sorteada e o texto sem precisar "recarregar" o site todo. É instantâneo!
- **Interface Mística (UX):** Um design pensado nas cores do tema, garantindo uma imersão agradável tanto num celular quanto no PC.

## 🛠️ Tecnologias Utilizadas

- **JavaScript (Vanilla):** O verdadeiro "cérebro" da página. Responsável pelas listas (Arrays), por embaralhar os dados e manipular a tela na hora do clique.
- **HTML5:** Toda a estrutura sólida das cartas e caixas de texto.
- **CSS3:** Todo o sombreamento agradável de botões, adaptação para smartphone (responsividade) e atmosfera visual.

## 💡 Quer testar a sua sorte na sua própria máquina?

Assim como um baralho de verdade, basta abrir a caixa para jogar. Não requer nenhuma instalação pesada, siga os passos abaixo:

1. No repositório, clique no botão verde `Code` e selecione `Download ZIP`.
2. Extraia (descompacte) os arquivos em qualquer lugar do seu computador.
3. Dê um duplo clique no arquivo `index.html`. O Oráculo carregará imediatamente no seu navegador principal. Que a sorte esteja com você!
